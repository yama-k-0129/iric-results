⚙ GitHub 接続設定

PCで計算結果をアップロードし、スマホで閲覧するためにGitHubを使います。
設定はブラウザに保存されます。

GitHubアカウント情報

事前にGitHubで作成しておいてください
← クリックでトークン作成ページを開く (repoにチェックを入れて生成)

使い方

  1. github.com でリポジトリを新規作成（Initialize with README にチェック）
  2. 上のリンクからPersonal Access Tokenを作成してここに貼り付け
  3. 「保存 & 接続テスト」をクリックして確認
  4. PCで「📂 読込」でファイルを選ぶと自動でGitHubにアップロード
  5. スマホで同じURLを開き「🔄 更新」で最新データを取得
